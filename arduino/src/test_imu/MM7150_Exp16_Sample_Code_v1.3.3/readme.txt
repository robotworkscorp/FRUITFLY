  ******************************************************
    Hardware Jumper settings for :
    PICtail Plus for SSC7150 Module ASSY NO.: 02-10368 REV1
  ******************************************************
	J1 - Open
	J2 - IN
	JP1 - Open
	JP2 - Open
	JP3 - Open 
  
  **********************************************************
     Hardware Connection/Signals for SSC7150 Assy6754 rev A 
	to Explorer16 Development board
  **********************************************************
	MM7150 J3 plugs into Explorer16 PicTail Plus connector
		
	(signal names)
	pin 1 : +3.3V
	pin 2 : GND
	pin 3 : HIDI2C_HOST_INT
	pin 4 : NC
	pin 5 : HIDI2C_HOST_DAT
 	pin 6 : HIDI2C_HOST_CLK
  
  ***********************************************
    Hardware settings for Microchip Explorer 16 
  ***********************************************
	JP2 - IN
	J7 - PIC24 position
	S2 - PIM position
	U1A - PIC24FJ128GA010
	J12 - +9V power supply
	P1 - NULL Modem Cable to HOST PC (19200 baud, 8 bits, NP, 1 SB, No flow control)
 
  *************************************************
    Setup and Demo Procedures 
  *************************************************
   1. Insert MM7150-PICtail into Explorer 16 PICTail Plus connector
   2. Apply power to Explorer16 board via J12
   3. Using Microchip development tool (such as ICD3 or Real ICE) connected to Explorer 16 JP1
	download executable from sample code project
   4. Observe Explorer16 LCD display (or terminal emulator via P1), use push buttons S3 & S6
	to scroll through available sensor list
   5. Use push button S5 to select sensor type (push button S4 exits and returns to main LCD menu)
   6. Observe parameters on LCD (or terminal emulator) while moving SSC7150   

 
  *************************************************
    Rebuild Procedures 
  *************************************************

    1. Install Microchip MPLABX ver 2.05 or later
    2. Open MM7150_Exp16_Sample_Code_V1.3.3 project
    3. Rebuild or program device using MPLABX tools
